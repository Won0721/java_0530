자동로그인

1. jsp에 자동로그인 체크박스
2. 컨트롤러에서 로그인 성공하면 회원정보에 자동로그인 체크여부를 수정

3-0 MemberVO에 me_session_ide, me_session_limit
3. 로그인 인터셉터에서 자동로그인이 체크되어 있으면 쿠키를 생성하고, 쿠키에 값(세션아이디)를 넣어
화면에 전달하고, DB에 세션아이디와 세션만료기간을 수정
4. 자동로그인인터셉터를 추가
 - 이미 로그인 되어 있으면 건너뜀
 - 로그인 되어있지 않으면 쿠키가 있는지 확인 
 - 쿠키가 있으면 쿠키에 있는 값과 일치하는 회워정보가 있는지 확인하고 가져옴
 - 가져온 회원정보가 있으면(자동로그인 성공) 세션에 회원정보를 추가 
 

5. 자동로그인 인터셉터를 등록, 적용(servlet-context.xml)

댓글등록
1. 댓글VO, 댓글컨트롤러, 댓글서비스/임플, 댓글다오/매퍼를 생성 및 연결
2. 게시글 상세 화면에서 댓글 등록화면을 구성
3. 댓글등록버튼을 클릭하면 ajax로 서버에 댓글정보를 전송
4. 컨트롤러에서 화면에서 보낸 데이터를 이용하여 댓글을 등록하고 등록 여부를 화면에 전송
5. 서버에서 보낸 댓글 등록여부를 이용하여 알림을 띄움 

댓글조회
1. 댓글 조회 화면을 구성(샘플)
2. ajax를 이용하여 현재 댓글 페이지에 맞는 댓글리스트와 페이지네이션을 요청
 - 컨트롤러 매서드 추가

3. 요청한 리스트와 페이지네이션을 이용하여 화면을 구성
